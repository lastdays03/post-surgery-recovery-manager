# 🔮 [미래 비전] 하이브리드 AI 온보딩 및 정밀 의학 프로파일링 계획

본 문서는 진단서, 소견서 등 의료 문서를 AI로 분석하는 **OCR 기술**과 사용자와의 **대화형 인터페이스(Chat)**를 결합하여, 사용자의 건강 상태를 가장 정밀하게 파악하고 최적의 회복 식단을 생성하는 **하이브리드 온보딩 비전**을 정의합니다.

## 1. 정밀 프로파일링 핵심 데이터 지표
식단 추천의 의학적 근거가 되는 핵심 지표들을 수집합니다. 모든 데이터는 '기본'과 '고급'으로 구분되어 관리됩니다.

### A. 기본 프로파일 (Required)
- **수술 정보**: 수술 종류(프로토콜 매핑), 수술 날짜(D-Day)
- **신체 지표**: 연령, 성별, 키, 몸무게 (BMI 자동 계산)
- **건강 상태**: 소화 능력(Good/Moderate/Poor), 기저질환(Comorbidities), 알러지(Allergies)

### B. 고급 의학 지표 (Selective)
- **영양 위험도**: NRS-2002 점수(≥3점 위험), SGA 등급(C등급 중증), 혈청 알부민 수치(<30 g/L 고위험)
- **대사 스트레스**: 6개월 내 체중 변화율(10~15% 감소 시 중증), 예상 금식 기간, 현재 섭취율
- **기능적 지표**: 위배출 지연 여부, GERD(역류성 식도염), 근감소증(Sarcopenia) 유무

## 2. 하이브리드 온보딩 프로세스 흐름
사용자는 자신에게 가장 편한 방식을 선택하여 정밀 프로파일을 완성할 수 있습니다.

### [Phase 1] 진입점 선택 (Onboarding Entry)
사용자는 온보딩 시작 시 두 가지 경로 중 하나를 선택합니다.
- **경로 A. 진단서로 빠르게 시작**: 문서 촬영/업로드를 통해 AI가 정보를 자동 추출합니다.
- **경로 B. 직접 입력하기**: 기존의 챗봇 대화 형식을 통해 정보를 단계별로 입력합니다.

### [Phase 2] 데이터 수집 및 추출 (Collection & Extraction)
- **문서 기반**: Google Document AI, OpenAI Vision 등 최적의 OCR 엔진이 문서를 스캔하여 지표를 추출합니다.
- **수동 기반**: 챗봇이 사용자의 답변을 분석하여 데이터 필드에 매핑합니다. 12종의 수술 프로토콜에 따른 맞춤형 질문이 이어집니다.

### [Phase 3] 추출 결과 검토 및 보완 (Review & Supplement)
- **스마트 리뷰**: AI가 추출한 데이터와 그 추출 근거(Confidence)를 사용자에게 보여줍니다. 사용자는 오검출된 내용을 즉시 수정할 수 있습니다.
- **누락 데이터 보충**: 문서에서 찾지 못한 필수 정보(예: 현재 식사 가능 여부 등)는 AI가 채팅으로 자연스럽게 질문하여 빈칸을 채웁니다.

### [Phase 4] 고급 프로파일링 활성화 (Advanced Profiling)
- **자동 감지**: 문서에서 알부민 수치나 NRS 점수가 감지되면 자동으로 '고급 모드'가 활성화됩니다.
- **선택적 입력**: 수동 입력 사용자의 경우, 더 정밀한 식단을 원하는 경우에만 추가 의학 지표 입력 단계를 진행할 수 있습니다.

## 3. 핵심 원칙 및 거버넌스
- **안전 최우선**: 알러지(`allergies`) 필드는 기저질환과 분리하여 식단 생성 시 'Hard-Filter'로 작동합니다.
- **데이터 보존**: AI가 판단하기 어려운 희귀 질환명 등은 원문 그대로 보존하여 전문 영양사 역할을 하는 식단 AI에게 전달합니다.
- **보안 및 프라이버시**: 모든 의료 데이터(PHI)는 전송/저장 시 암호화되며, 분석 과정에서 개인 식별 정보는 마스킹 처리됩니다.

---
> [!IMPORTANT]
> 본 하이브리드 비전은 **"문서 한 장으로 끝내는 편리함"**과 **"대화로 풀어가는 꼼꼼함"**을 동시에 제공하여, 수술 후 환자가 오직 '회복'에만 전념할 수 있는 환경을 만드는 것을 목표로 합니다.

| 구분 | 데이터 필드 | 기준 및 의학적 의미 |
| :--- | :--- | :--- |
| **수술 정보** | 수술 종류 | GI(위·대장·췌장), 정형, 두경부, 이식, 비만수술 등 (프로토콜 결정) |
| | 수술 날짜 | D-Day 기준 회복 및 식단 단계 계산의 기점 |
| | 수술 범위/침습도 | 대수술 vs 경수술 (대사 스트레스 강도 판단) |
| **영양 위험도** | NRS-2002 점수 | ≥3점 위험, ≥5점 고위험 (영양 집중 지원 필요성 판단) |
| | 체중 변화 | 6개월 내 10~15% 감소 시 중증 위험 (영양 불량 상태) |
| | BMI | 18.5 미만 시 고위험 (영양 저장량 부족) |
| | SGA 등급 | C등급일 경우 중증 영양 불량으로 간주 |
| | 혈청 알부민 | 30 g/L 미만 시 고위험 (단백질 결핍 및 부종 위험) |
| **대사/질환** | 기저질환 | 당뇨, 고혈압, 간·신장 질환 (탄수/단백·전해질 조절의 근거) |
| | 소화기 기능 | 위배출 지연, GERD (탄수화물 음료 및 식단 종류 제한) |
| **섭취 능력** | 경구 가능 여부 | 입으로 음식 섭취가 가능한지(Yes/No) 여부 |
| | 예상 금식 기간 | >5일, >7일, >14일 (영양 공급 경로 및 강도 결정) |
| | 섭취율 | 필요량의 50% 미만 섭취 여부 (추가 보충제 필요성) |
| **연령/근감소** | 연령 | 고령자일 경우 더 높은 단백질 권장량 적용 |
| | 근실실(Sarcopenia) | 근감소증 존재 시 합병증 위험 증가 및 재활 강도 조정 |
